# js-code-smells-dataset-metrics-and-embeddings

## Class datasets
* classes.csv contains a "merged" dataset with both embedding and metrics for each class code snippet
* classes_embeddings.csv contains a dataset with only embeddings each class code snippet
* classes_metrics.csv contains a dataset with only metrics for each class code snippet

## Method datasets
* methods.csv contains a "merged" dataset with both embedding and metrics for each method code snippet
* methods_embeddings.csv contains a dataset with only embeddings each method code snippet
* methods_metrics.csv contains a dataset with only metrics for each method code snippet
